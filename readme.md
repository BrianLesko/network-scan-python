
# Network Scan for LAN networks in Python
discover other devices on a LAN network, along with your own IP, subnet, and the broadcast address. 
Written to work on Apple Computers only. 

&nbsp;

<div align="center"><img src="docs/preview.gif" width="800"></div>

&nbsp;

## Software Dependencies

This code uses the following libraries:
- `netifaces`: for retrieving the subnet only

&nbsp;

## Hardware Dependencies
- `An Apple or Mac Computer`

## Usage
```
pip install netifaces
python app.py
```
